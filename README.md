## 概要(Description)
この課題は42cursusの課題です。get_nextline関数を実装します。get_next_line関数とは、ファイルディスクリプタを引数として受け取り、呼び出すごとにファイルディスクリプタから一行ずつを読み取って返す関数です。ファイルの読み書きと静的変数を理解して使用できるようになるなどの事や自作ライブラリにprintf関数を実装する事を目的としています。

## 指示(Instructions)
全てのソースファイルは、`-Wall -Wextra -Werror` フラグを用いて `cc` でコンパイルされます。
`cc -Wall -Wextra -Werror -D BUFFER_SIZE=42 <files>.c`のようにコンパイルします。
`-D`フラグによって、`BUFFER_SIZE`を指定できます。指定しない場合、`BUFFER_SIZE`は`42`になります。
